# AfriFutures - One Pager

**De-risking African Agriculture Through Autonomous Agent Trading**

---

## The Problem

African smallholder farmers lose **$120B annually** to:
- Price volatility (crops worth 60% less at harvest than at planting)
- Post-harvest losses (no buyer, no storage)
- Middlemen taking 40-60% of value
- No access to credit (no credit history)

**Current solutions fail because:**
- Banks won't lend without collateral
- Traditional insurance is too expensive (5-10% premiums)
- Farmers can't use complex DeFi protocols

---

## Our Solution

**AfriFutures** lets farmers hedge crop prices via WhatsApp in **30 seconds**—no app, no bank, no jargon.

```
Farmer: "Hedge 500kg coffee @ KSh 200/kg"

AfriFutures: "Done! You're protected if price drops.
               Premium: KSh 100 (~$0.85)"
```

**How it works:**
1. Farmer sends hedge request via WhatsApp
2. AI agents match with counterparties (hedgers, speculators)
3. Premium (0.5%) goes into peer pool
4. Chainlink oracle triggers automatic payout on price drop
5. Farmer receives USDC instantly—no bank needed

---

## How It Works (3 Flows)

### 1. Farmer Hedge (WhatsApp)
```
"Hedge [quantity][crop] @ [price]/kg"

Example: "Hedge 2t maize @ KSh 80/kg"

System responds:
- Strike price confirmed
- Premium calculated (0.5%)
- Peer pool contribution
- Position opened on-chain
```

### 2. Co-op Verification
```
Farmers join via co-op invite
↓
Co-op verifies: land docs, crop photos, harvest records
↓
Farmer gets DVC (Digital Verifiable Credential)
↓
DVC score unlocks: higher advance rates, lower premiums
```

### 3. AI Agent Trading
```
AI agents connect via API
↓
Agents see order book (commodity + price + quantity)
↓
Agents execute trades autonomously
↓
Platform takes 0.5% fee
↓
Revenue split: 70% farmers, 30% platform
```
